    Among all Windows errors, drive errors are the scariest ones. Your storage drive can contain lots of personal and professional files, and if it dies, you run the risk of data loss.

    One particular issue that several users have been facing is the 'Scan drive for errors' notification by the Security and Maintenance component of Windows. Since the notification itself does not specify which drive could be problematic or what might be causing the issue, finding the right solution can be a bit tricky.

    In this guide, we will have a look at the four most effective troubleshooting steps that are likely to help you identify the faulty drive and fix issues related to it in no time.

    1. Run a CHKDSK Scan

    The easiest and most effective way of fixing this notification on Windows is doing what it says; scanning the drive. To do this, we're going to need the aid of a CHKDSK (check disk) scan. This scan will scan all the drives in the system looking for potential issues that might be causing the issue at hand.

    If any problems are found on the disks, CHKDSK will fix them without needing a lot of user involvement. We have listed two methods of running a CHKDSK scan below. Proceed with the method you prefer.

    1. How to Run a CHKDSK Scan via Command Prompt

    You must be logged into Windows as an administrator to run it via Command Prompt.

    Here is what you need to do:

  • Press Win + R to open Run.
  • Type cmd in the text field of Run and press Ctrl + Shift + Enter together to open Command Prompt with administrative privileges.
  • Inside the Command Prompt window, type the command mentioned below and hit Enter to execute it. Make sure to replace x with the name of the drive you want to check. Since we will be checking the C: drive for errors, we will be replacing x with c.
    chkdsk /x:
  • Wait for the command to complete. It might take a while, so hang in there. If you receive a ‘The volume is in use by another process’ message, the scan will be scheduled to run at the next startup.
  • Once the command is executed, you can also check for bad sectors by executing the following command:
    chkdsk /r x:
  • After the command is executed, close Command Prompt and check if the issue is resolved.

    • 2. Run CHKDSK via File Explorer

    If you do not want to use Command Prompt, then you can take the simpler approach — running the CHKDSK utility via File Explorer.

    All that you need to do is access File Explorer and check the targeted disk for errors. Since we are not sure which drive might be causing the issue in the case of an error in consideration, we highly recommend you check each drive for potential issues.

    It is going to be a lengthy process, but the scans will identify and fix the problems that are found in drives, thus resolving the error in the process.

    2. Run a SMART Test

    The error code might also be notifying you about a failing hard drive. To check if this is the case, you can run a SMART (Self-Monitoring, Analysis, and Reporting Technology) test via Command Prompt.

    This test will display statuses like OK, Bad, and Unknown after you run the test. The next troubleshooting strategies will depend on this status.

    Follow these steps to proceed:

  • Type cmd in Windows search and click on Run as administrator.
  • Click Yes in the UAC prompt.
  • In the following window, type the command mentioned below and hit Enter.
    wmic diskdrive get status
  • Now check the drive statuses. If all is good, then the issue is system-related. However, if the status is bad or unknown, then it is a drive-related issue.

    • You can also check if your hard drive is failing by connecting to another PC. If the error pops up there as well, then it's time to get it checked by a professional.

    3. Run a Malware Scan

    If the CHKDSK scan did not find an issue, then the next thing that you should do is scan your system for malware and viruses. If you have an antivirus program installed, then proceed with running a scan using it.

    The exact steps of running a scan in different antivirus programs will differ. We will be proceeding with Avast antivirus to give you an idea about how to run a malware scan.

    Follow these steps to proceed:

  • Launch your antivirus program and head over to the Protection section.
  • Click on Virus Scans.
  • In the following window, hit the Run Smart Scan button. Wait for the scan to proceed and then follow the on-screen instructions to finish.

    • If you do not have third-party antivirus software, you can consider checking our guide on ways to scan for viruses without buying antivirus software.

    4. Perform a System Restore

    Another way of fixing annoying errors is by using the built-in System Restore feature.

    The System Restore utility takes snapshots of the working state of your system before performing critical operations, which are stored as the restore points. As the name suggests, these restore points allow you to revert to a previous system state if anything goes wrong.

    There is a chance that you are facing a drive error because of a faulty update or a corrupt app. In this method, we will try to revert your system to a state where the error at hand was not present.

    Here is what you need to do:

  • Type Create a restore point in Windows search and click Open.
  • In the following dialog, head over to the System Protection tab and click on the System Restore button.
  • You should now see a list of restore points available. Choose one (preferably the most recent one) and click Next.
  • Follow the on-screen instructions to proceed and once your system is restored to a previous point in time, check if the error is resolved.

    • Drive Issues, Now Fixed

    You now know how to fix the 'Scan drive for errors' notification on Windows. Drives are an essential component of all operating systems, so keeping them error-free is important. To avoid such issues in the future, be sure your system is updated, and you have a reliable security program that can detect malware instantly.
